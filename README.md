# Trivia Quiz System - C++17

## Overview

A professional console-based trivia quiz system developed in C++17.

The project demonstrates:

- Object-Oriented Programming
- Inheritance
- STL vectors
- File handling
- Operator overloading
- Template functions
- Bidirectional communication between applications

---

# Applications

## Player Application

Features:
- Play trivia quiz
- Save progress
- Resume game
- View leaderboard
- View questions

## Admin Application

Features:
- Add questions
- Delete questions
- View leaderboard
- Reset progress
- Admin authentication
